【回答】：斐波那契数列的定义是：前两个数字是0和1，后续每个数字都是前面两个数字之和。

```python
def fibonacci(n):
    if n <= 0:
        return "输入的数值必须大于0"
    elif n == 1:
        return 0
    elif n == 2:
        return 1
    else:
        fib_sequence = [0, 1]
        for i in range(2, n):
            fib_sequence.append(fib_sequence[i-1] + fib_sequence[i-2])
        return fib_sequence[n-1]
```

    a, b = 0, 1
    for i in range(n):
        fib_sequence.append(a)
        a, b = b, a+b
    return fib_sequence[:m] if m < n else fib_sequence
